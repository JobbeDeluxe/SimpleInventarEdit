# SimpleInventarEdit – Patchnote v2.4.1

## Highlights
- Offline gespeicherte Inventare werden jetzt auch geladen, wenn sie in älteren `offline-data.yml`-Strukturen auf Root-Ebene abgelegt sind.
- Bestehende Unterstützung für moderne `players`-Sektionen bleibt erhalten und verhindert so Datenverlust bei gemischten Speicherformaten.

## Verbesserungen
- Fehlerausgaben beim Laden der Offline-Daten wurden geschärft, damit inkonsistente oder fehlerhafte Einträge schneller auffallen.

## Hinweise
- Funktionalität und Bedienung im Spiel bleiben unverändert; alle Änderungen zielen auf bessere Rückwärtskompatibilität ab.
- Getestet auf Paper 1.21.4 mit Java 17+.
